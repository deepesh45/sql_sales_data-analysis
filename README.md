# Sales Data Analysis using SQL

## Project Overview
This project demonstrates SQL-based data analysis using relational datasets.

## Tools Used
- SQL
- Oracle Live SQL

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- JOINS
- Aggregate Functions
- Subqueries

## Analysis Performed
- Employee salary analysis
- Department-wise insights
- Customer and sales trends
- Aggregate reporting
